    **Step 1: Add `getPublicLinkBySlug` to queries.ts** (`src/lib/supabase/queries.ts`)

    Add a new exported function `getPublicLinkBySlug(supabase: SupabaseClient, slug: string)` to the existing queries file. This function queries the `links` table filtered by `slug` and `is_active = true`, with a join on the `files` table to get file name, mime_type, and size_bytes. Use `supabaseAdmin` from `@/lib/supabase/admin.ts` as the client passed to this function. The admin client (service role) bypasses RLS, which is safe because this function is only called from a server component -- the service role key never reaches the client.

    **Step 2: Create the public buyer-facing page** (`src/app/l/[slug]/page.tsx`)

    This is a **dynamic server component** -- no "use client".

    **Next.js 16 important:** The `params` prop is a `Promise`. Must use `const { slug } = await params`.

    **generateMetadata function:**
    - Fetch link via `getPublicLinkBySlug(supabaseAdmin, slug)`
    - If link not found or not active: return `{ title: "Not Found" }`
    - If found: return metadata with:
      - `title`: link.title
      - `description`: link.description or "Get this file on Unlockt"
      - `openGraph.title`: link.title
      - `openGraph.description`: link.description or "Get this file on Unlockt"
      - `openGraph.images`: If `preview_url` exists, include it as `[{ url: link.preview_url }]`
      - `openGraph.type`: "website"
      - `twitter.card`: "summary_large_image" (if preview) or "summary" (if no preview)

    **Default page component:**
    - Fetch link via `getPublicLinkBySlug(supabaseAdmin, slug)` (yes, this queries twice -- once in generateMetadata, once in page. Wrap the fetch in a `cache()` function from React to deduplicate).
    - If link not found: call `notFound()` from `next/navigation` (renders the closest not-found.tsx)
    - If link found but `is_active === false`: call `notFound()`
    - If link found and active: render the page layout

    **Page layout:**
    - Full-height centered layout: `min-h-screen flex items-center justify-center p-4`
    - Background: use the default dark background from the app
    - Content: Render `<LinkPageCard link={linkData} />`
    - Below the card: Small "Powered by Unlockt" link back to `/` (subtle, text-muted-foreground)

    **Important considerations:**
    - This page has NO auth requirement -- it's fully public
    - Uses admin client (service role) to bypass RLS, which is safe because this is a server component
    - The admin client is imported from `@/lib/supabase/admin.ts` which already exists in the codebase
