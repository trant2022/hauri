  <files>

  <action>
    1. Install stripe package: `npm install stripe`

    2. Create database migration `supabase/migrations/00003_transactions_rls_and_rpcs.sql`:
       - ALTER TABLE public.transactions ADD COLUMN stripe_payment_intent_id text
       - CREATE INDEX idx_transactions_payment_intent ON public.transactions (stripe_payment_intent_id)
       - CREATE RLS policy "Users can read own transactions" on public.transactions FOR SELECT TO authenticated USING (link_id IN (SELECT id FROM public.links WHERE user_id = auth.uid()))
       - CREATE OR REPLACE FUNCTION increment_unlock_count(link_id_param uuid) RETURNS void LANGUAGE sql AS $$ UPDATE public.links SET unlock_count = unlock_count + 1 WHERE id = link_id_param; $$
       - CREATE OR REPLACE FUNCTION decrement_unlock_count(link_id_param uuid) RETURNS void LANGUAGE sql AS $$ UPDATE public.links SET unlock_count = GREATEST(unlock_count - 1, 0) WHERE id = link_id_param; $$

    3. Apply the migration to local Supabase: `npx supabase db push` (or if using local dev, `npx supabase migration up`)
       If supabase CLI commands fail (no local instance), that's fine -- the migration file is the deliverable.

    4. Create `src/lib/stripe/client.ts`:
       - Export const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!, { typescript: true })
       - Import Stripe from "stripe"

    5. Create `src/lib/stripe/checkout.ts`:
       - Import stripe from ./client, calculateFees from @/lib/fees
       - Export async function createCheckoutSession({ linkId, slug, linkTitle, priceAmountCents, currency })
       - Calculate fees with calculateFees(priceAmountCents)
       - Determine payment_method_types: ["card", "twint"] if currency.toLowerCase() === "chf", else ["card"]
       - Create session with stripe.checkout.sessions.create():
         - mode: "payment"
         - payment_method_types
         - line_items: single item with price_data (currency, unit_amount = fees.totalBuyerPays, product_data.name = linkTitle), quantity 1
         - metadata: { link_id: linkId, base_price: String(priceAmountCents), platform_fee: String(fees.platformFee), creator_amount: String(fees.creatorReceives) }
         - success_url: `${process.env.NEXT_PUBLIC_APP_URL}/l/success?session_id={CHECKOUT_SESSION_ID}`
         - cancel_url: `${process.env.NEXT_PUBLIC_APP_URL}/l/${slug}`
       - Return session.url (the redirect URL)

    6. Update `src/types/database.ts`:
       - Add stripe_payment_intent_id: string | null to transactions Row type
       - Add stripe_payment_intent_id?: string | null to transactions Insert type
       - Add stripe_payment_intent_id?: string | null to transactions Update type
       - Add Functions section with increment_unlock_count and decrement_unlock_count declarations:
         ```
         Functions: {
           increment_unlock_count: {
             Args: { link_id_param: string }
             Returns: undefined
           }
           decrement_unlock_count: {
             Args: { link_id_param: string }
             Returns: undefined
           }
         }
         ```
  </action>

       - If invalid: return NextResponse.json({ error: "Link not available" }, { status: 400 })

       - Return NextResponse.json({ url: checkoutUrl }) with the Stripe Checkout redirect URL
       - Wrap in try/catch, return { error: string } on failure with 500 status
